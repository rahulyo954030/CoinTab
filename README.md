<h3>Step 1: Install node_modules for React App</h3>

```Javascript
npm install
```
<br/>

<h3>Step 2: Start the React app</h3>

```Javascript
npm start
```
<br/>

<h3>Step 3: Install node_modules in Server folder</h3>

```Javascript
npm install
```
<br/>

<h3>Step 4: Start the backend server</h3>

```Javascript
npm run dev
```
