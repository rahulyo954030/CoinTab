<h3>Step 1: Install node_modules in client folder</h3>

```Javascript
npm install
```
<br/>

<h3>Step 2: Start the react app</h3>

```Javascript
npm start
```
<br/>

<h3>Step 3: Install node_modules in server folder</h3>

```Javascript
npm install
```
<br/>

<h3>Step 4: Start the backend server</h3>

```Javascript
npm run dev
```
